# test/e2e

E2E тесты против поднятого Vite-приложения.

## Файлы

- `campaign.spec.ts` — проверка кампанийного потока: главная -> карта -> вход в зону.
- `trainer.spec.ts` — проверка тренажёра: запуск механики и завершение 12-задачной сессии.

## Конфиг

Параметры запуска браузера/сервера находятся в `playwright.config.ts`.
