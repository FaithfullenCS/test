# src

Основной код приложения `Cash Flow Quest RU`.

## Как устроено

1. `main.tsx` монтирует React-приложение и подключает глобальные стили.
2. `App.tsx` поднимает `GameProvider` и роутинг.
3. `state/GameContext.tsx` хранит игровой стейт, очереди тренажёра, начисление LP и сохранение в `localStorage`.
4. `pages/*` реализуют экранные сценарии (кампания, тренажёр, кейсы, профиль, итоги).
5. `lib/*` содержит чистую игровую логику, селекторы и storage-слой.
6. `data/*` содержит статические данные мира и генерацию заданий.

## Подпапки

- `components/` — переиспользуемые UI-блоки.
- `data/` — контент мира, зон, заданий, кейсов, ARC/Sprint-наборов.
- `lib/` — движок начисления/проверки и вычисляемые селекторы.
- `pages/` — маршрутизируемые страницы.
- `state/` — глобальный контекст игры.
- `test/` — setup для Vitest.
- `types/` — типы доменной модели.

## Ключевой поток данных

`data -> GameContext -> pages/components -> submit -> engine/storage -> обновлённый progress`.
