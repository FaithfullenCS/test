# src/pages

Маршрутизируемые экраны приложения.

## Файлы

- `HomePage.tsx` — каталог миров и быстрый вход в кампанию/тренажёр.
- `WorldMapPage.tsx` — интерактивная карта зон с выбором узла и деталями прогресса.
- `ZoneIntroPage.tsx` — обзор зоны перед прохождением.
- `ChallengePage.tsx` — универсальный экран задания для кампании и тренажёра.
- `CasePage.tsx` — 3-шаговый режим `Case Ladder`.
- `TrainerPage.tsx` — выбор механики/сложности/расширенных режимов.
- `ProfilePage.tsx` — сводная статистика игрока.
- `ResultsPage.tsx` — финальный экран с сертификатом и медалями.

## Ключевая роль

Страницы связывают UI с `GameContext`: читают прогресс и вызывают submit/record-методы.

## Тесты

Для основных страниц есть unit-тесты в этой же папке (`*.test.tsx`).
