# src/components

Переиспользуемые UI-компоненты без собственного доменного состояния.

## Файлы

- `AppShell.tsx` — общий каркас приложения: topbar со статистикой, навигация, контейнер для страниц.
- `WorldMapCard.tsx` — карточка узла зоны на карте мира (иконка района, блокировка, медаль, выбор).
- `WorldMapPage.test.tsx` — тесты взаимодействия с картой через страницу `WorldMapPage`.

## Принцип

Компоненты получают данные сверху (из страниц/контекста), а наружу отдают только события (`onSelect` и т.п.).
