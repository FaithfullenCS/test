# src/types

Типы доменной модели игры.

## Файлы

- `game.ts` — единая схема типов:
  - зоны, механики, сложности,
  - структуры заданий,
  - профиль игрока и статистика,
  - контракты storage.

## Зачем это нужно

Все ключевые слои (`data`, `lib`, `state`, `pages`) используют общую типовую модель, чтобы сохранять согласованность бизнес-логики.
