# skills

Локальные skill-пакеты проекта, описывающие игровые механики и режимы.

## Что здесь есть

- `cfq-term-forge-mechanic`
- `cfq-sentence-builder-mechanic`
- `cfq-context-choice-mechanic`
- `cfq-boardroom-boss-mechanic`
- `cfq-adaptive-recall-cycle`
- `cfq-case-ladder-mode`
- `cfq-liquidity-sprint-mode`
- `cfq-difficulty-balancing`
- `cfq-scoring-and-progression`
- `cfq-world-map-layout`
- `cfq-world-creation`

## Структура skill-папок

- `SKILL.md` — инструкция по использованию навыка.
- `agents/openai.yaml` — метаданные агента.
- `references/*` — спецификации (есть не во всех skill).

## Примечание

Часть `SKILL.md` уже детализирована (например, `term_forge`, `sentence_builder`, `context_choice`, `boardroom_boss`), часть пока содержит TODO-шаблон.
