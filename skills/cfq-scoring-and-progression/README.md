# cfq-scoring-and-progression

Правила начисления LP и прогрессии по зонам/наградам.

## Где реализовано в приложении

- Скоринг и прогрессия: `src/lib/engine.ts`.
- Применение в state: `src/state/GameContext.tsx`.
- Визуализация прогресса: `src/components/AppShell.tsx`, `src/pages/ResultsPage.tsx`, `src/pages/ProfilePage.tsx`.

## Ключевые правила

- LP за попытки: `10 / 6 / 3` (+ модификатор сложности).
- Подсказка уменьшает награду на penalty сложности.
- Бонус серии: `+8 LP` каждые 5 подряд верных.
- Медали зоны: `bronze >=60`, `silver >=80`, `gold >=92`.
- Разблокировка следующей зоны: завершение предыдущей + точность `>=70%`.
